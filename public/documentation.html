MyFLIX API

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.
Action Method Query Parameters Endpoint URL Response
Allow new users to register POST JSON object with user data:

Username: "TestUser"",

Password: "TestPassword",

Email: "Test@gmail.com"

DOB: "2001-01-01"

Return a list of all movies in the database GET None passed (retrieve all) '/movies' Returns a JSON array of all movies in the database:

MovieID: Integer,

Title: String,

Genre: String,

Description: String

Actors: String Array

MovieID: Integer,

Title: String,

Genre: String,

Return description of a director by DIRECTOR NAME GET Director '/directors/:director' Returns a JSON object containing all movies in the database by the name of the director name that was passed in the url:

Allow users to update their user info BY USERNAME PUT Username '/users/:username' JSON object with updated user data. Example:

Allow users to add a movie to their list of favorite movies POST Username, MovieID '/users/:username/movies/:movieID' Returns a JSON object with updated user data. Example:

UserID: "xxxx",

Username: "TestUser"",

Password: "UpdatedPassword",

Email: "Test@gmail.com"

"FavoriteMovies": ["61d33b12ce15f2ca5a44c03c"]

DOB: "2001-01-01"

Allow users to remove a movie from their list of favorite movies DELETE Username, MovieID '/users/:username/movies/:movieid' Returns a JSON object with updated user data. Example:

UserID: "xxxx",

Username: "TestUser"",

Password: "UpdatedPassword",

Email: "Test@gmail.com"

"FavoriteMovies": ["61d33b12ce15f2ca5a44c03c"]

DOB: "2001-01-01"

Allow existing users to deregister DELETE Username '/users/:username' Returns a text confirming that the user's email was deleted successfully.
Return a list of all actors in the database GET None passed (retrieve all) '/actors' Returns a JSON objects of all actors in the database. Example:

ActorID: "61d3324a3beaf479d1a06aa4",

Name: "Song Kang-ho",

Bio: ""Song Kang-ho is a South Korean actor."

Birth: 1967-01-17
Return data about an actor (bio, birth year, death year) by ACTOR NAME GET Actor 'actors/:actorName' Returns a JSON object containing data about an actor based on the actor name that was passed in the url:

ActorID: Integer,

Name: String,

DOB: String,

DOD: String
