MyFLIX API

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Username: String,

Movies: String Array

Action Method Query Parameters Endpoint URL Response
Return a list of all movies in the database GET None passed (retrieve all) '/movies' Returns a JSON array of all movie objects in the database:

MovieID: Integer,

Title: String,

Genre: String,

Description: String

Actors: String Array

Return data about a single movie BY TITLE GET Title '/movies/:title' Returns a JSON object with data on a single movie based on the title passed in the url:

MovieID: Integer,

Title: String,

Genre: String,

Description: String

Actors: String Array

Return a list of movies by GENRE (e.g., “Thriller”) GET Genre '/movies/:genre' Returns a JSON array of all movies in the database based on the genre passed in the url:

MovieID: Integer,

Title: String,

Genre: String,

Description: String

Actors: String Array

Return data about a director (bio, birth year, death year) BY DIRECTOR NAME GET Director '/movies/directors/:directorName' Returns a JSON object containing data of the director based on the name that was passed in the url:

DirectorID: Integer,

Name: String,

DOB: String,

DOD: String
Return a list of movies BY DIRECTOR NAME GET Director '/movies/:directorName' Returns a JSON object containing all movies in the database by the name of the director that was passed in the url:

MovieID: Integer,

Title: String,

Genre: String,

Description: String

Actors: String Array

Return data about a actor (bio, birth year, death year) BY ACTOR NAME GET Actor '/movies/actors/:actorName' Returns a JSON object containing data of the actor based on the name that was passed in the url:

ActorID: Integer,

Name: String,

DOB: String,

DOD: String
Return a list of movies BY ACTOR NAME GET Actor '/movies/:ActorName' Returns a JSON object containing all movies in the database by the name of the actor that was passed in the url:

MovieID: Integer,

Title: String,

Genre: String,

Description: String

Actors: String Array

Retrun a list of favorite movies by USER NAME GET Username '/users' Returns a JSON object containing the list of favorite movies by the username passed in the URL in the url:

MovieID: Integer,

Title: String,

Genre: String,

Description: String

Actors: String Array

Allow new users to register POST JSON object with user data:

UserID: Integer,

Username: String,

Password: String,

Email: String

DOB: String

'/users' Returns success message with username added.
Allow users to update their user info (username) PUT Username '/users/:username' Returns success message with the username that was updated by the user.
Allow users to add a movie to their list of favorite movies POST JSON object with movie data:

MovieID: Integer,

Title: String,

Genre: String,

Director: String

Actors: String Array

'/users/add/:movieName' Returns a text confirming which movie was added successfully.
Allow users to remove a movie to their list of favorite movies DELETE Title '/users/remove/:movieName' Returns a text confirming which movie was removed successfully.
Allow existing users to deregister DELETE Username '/users/deleteAccount' Returns a text confirming that the user's email was deleted successfully.
